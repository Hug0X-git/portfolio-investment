# Portfolio Investments

## Abstract

This project, developed as part of a group assignment at **ESCP Business School**, presents the design and implementation of a multi-asset portfolio investment platform. The objective is to integrate financial theory with web-based technologies in order to provide an interactive environment for portfolio construction, optimization, and risk analysis. The platform incorporates investor profiling, quantitative analysis tools, and data visualization features, alongside export functionalities enabling further analysis in Python. Selected components were developed with the assistance of **Claude** and **CODEX**.

## Introduction

The increasing accessibility of financial markets has heightened the need for tools that enable informed portfolio management. This project aims to address this need by combining principles of modern portfolio theory with user-friendly web development. It demonstrates how financial modelling, data analysis, and interface design can be integrated into a coherent digital solution.

The platform allows users to construct and evaluate investment portfolios within a structured analytical framework, while maintaining accessibility for non-expert users.

## Functional Overview

The application provides a range of functionalities designed to guide users through the investment decision-making process:

- **Investor Profiling**: Users complete a risk assessment questionnaire to determine their risk tolerance and investment profile.
- **Asset Universe Configuration**: Selection and customization of a diversified set of financial assets.
- **Portfolio Construction and Optimization**: Implementation of optimization techniques to identify efficient asset allocations.
- **Risk and Correlation Analysis**: Evaluation of portfolio risk through statistical indicators and correlation matrices.
- **Data Visualization**: Graphical representation of asset allocation and portfolio characteristics.
- **Target Allocation Display**: Presentation of optimized portfolio weights.
- **Python Export Functionality**: Generation of outputs compatible with Python for extended quantitative analysis.

## Methodological Framework

The project draws upon key concepts from:

- **Modern Portfolio Theory (Markowitz, 1952)**
- **Quantitative Finance and Statistical Analysis**
- **Data Visualization Techniques**
- **Web Application Development**

The integration of these domains enables both analytical rigor and practical usability.

## Technical Implementation

The platform is implemented using standard web technologies, ensuring accessibility and ease of deployment:

- **HTML** for structural design  
- **CSS** for styling and layout  
- **JavaScript** for interactivity and data processing  
- **Vercel** for deployment and hosting  

## Project Context

This work was conducted as part of a collaborative academic project at **ESCP Business School**. It reflects an interdisciplinary approach combining finance, programming, and user experience design. Certain technical and creative aspects were developed with the support of **Claude**, contributing to the overall functionality and design of the platform.
